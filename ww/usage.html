Commands #

!help

Provides basic help in Discord (which isn't really that helpful)

!rolelist

**This command is available for players with "Dead" role and MC(s) only.

**To prevent accidentally leaking the list out in game, this command can only be used in the #spectators channel.

Sends out role list. Example:

1. seer
2. village
3. wolf
4. village
5. idiot
6. wolf
7. witch

!endgame

**This command is available for MC(s) only.

Should only be used when game ends. Remove wolves' permission to view wolf discussion channel.
